    kobayashi_maru v0.0.1 API Reference

    Modules

    KobayashiMaru

    KobayashiMaru keeps the contexts that define your domain and business logic.

    KobayashiMaru.Accounts

    Accounts context that holds all the logic for accounts.

    KobayashiMaru.Accounts.User
    KobayashiMaru.Application
    KobayashiMaru.Auth.ErrorHandler

    The Guardian Pipeline will need a module to handle the error it may found.

    KobayashiMaru.Auth.Guardian

    This file responsibility is to convert between a token an a resource (a user) and between a resource (our user) and a unique identifier to be encoded in the token that will be passed all over the place.

    KobayashiMaru.Auth.Guardian.Plug
    KobayashiMaru.Auth.Pipeline

    Essentially checks that a request made to our app has a header with the token, ensure that the token is valid and, if it is, loads the resource automatically (that is, loads the user corresponding to the token's claims["sub"] value in the encrypted token).

    KobayashiMaru.Repo
    KobayashiMaruWeb

    The entrypoint for defining your web interface, such as controllers, views, channels and so on.

    KobayashiMaruWeb.Endpoint
    KobayashiMaruWeb.ErrorHelpers

    Conveniences for translating and building error messages.

    KobayashiMaruWeb.ErrorView
    KobayashiMaruWeb.Gettext

    A module providing Internationalization with a gettext-based API.

    KobayashiMaruWeb.ProductController
    KobayashiMaruWeb.Router
    KobayashiMaruWeb.Router.Helpers

    Module with named helpers generated from KobayashiMaruWeb.Router.

    KobayashiMaruWeb.SessionController

    Its responsibilities are to create new sessions, delete them and refresh them.

    KobayashiMaruWeb.SessionView

    This is similar although a bit more complex. The show.json view renders the user and the token to the client.

    KobayashiMaruWeb.UserController
    KobayashiMaruWeb.UserSocket
    KobayashiMaruWeb.UserView

    The render method receives a user and returns a map with the properties that we want to expos to the clients.
